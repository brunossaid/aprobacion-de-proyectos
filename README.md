# Aprobacion de Proyectos

Trabajo practico 2 - Proyecto de Software

Aplicación de consola + API REST desarrollada en **C#** para gestionar solicitudes de aprobación de proyectos.

## Estructura del proyecto

- `Domain`: entidades del dominio.
- `Application`: interfaces y servicios de aplicacion.
- `Infrastructure`: acceso a datos e implementacion de servicios.
- `ConsoleApp`: interfaz de usuario por consola.
- `API`: web API RESTful con Swagger (OpenAPI)
